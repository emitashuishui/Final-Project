# Property Rental Management System
This project demonstrates a 3NF normalized database built for Oracle.

## How to Run
To recreate the database environment, run the SQL files in the following order:
1. `sql/reset.sql` - Clears existing tables.
2. `sql/schema.sql` - Creates tables and constraints.
3. `sql/sample_data.sql` - Populates the database with test records.
